# CODEX_PROMPTS.md

这个文件存放可以直接发给 Codex 的短提示词。建议一次只复制一个。

## 第一次打开仓库后

```text
请先阅读 AGENTS.md、PROJECT_BRIEF.md、TASKS.md。只总结你理解到的项目目标、技术路线、目录结构和当前任务拆分，不要修改代码。
```

## 开始某个任务

```text
请阅读 AGENTS.md、PROJECT_BRIEF.md、TASKS.md，然后只完成 Task 00。完成后说明修改了哪些文件、如何验证、下一步该做什么。
```

把 Task 00 替换成 Task 01、Task 02 等。

## 检查当前进度

```text
请扫描当前仓库，按照 TASKS.md 的任务清单判断哪些已完成、哪些未完成。不要修改代码，只输出状态表和下一步建议。
```
