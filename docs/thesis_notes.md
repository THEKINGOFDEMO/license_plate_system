## 论文主题

《基于深度学习的车牌检测与识别系统设计与实现》

## 可写入论文的核心主线

### 1. 研究背景与意义

- 智能交通场景下车牌识别的应用价值；
- 传统方法在复杂场景下的局限；
- 基于深度学习的检测与识别方法具有更强鲁棒性。

### 2. 系统技术路线

```text
CCPD 数据集
  -> 数据清洗与标注解析
  -> YOLO 检测车牌区域
  -> CRNN 识别车牌字符
  -> Django 提供后端接口
  -> Vue 实现前端展示
```

### 3. 系统功能

- 图片上传；
- 车牌检测；
- 车牌裁剪；
- 字符识别；
- 结果可视化；
- 历史记录查询。
